# Détection de Commentaires Toxiques - Projet ML

## Contexte
Ce projet s'inscrit dans le cadre d'un challenge Kaggle visant à détecter les commentaires toxiques dans les forums. L'objectif est de construire un modèle de classification capable d'identifier automatiquement les commentaires toxiques tout en prenant en compte les biais potentiels liés à l'identité.

Source: [Kaggle Competition - Commentaires Forum Jigsaw](https://www.kaggle.com/competitions/commentaires-forum-jigsaw/overview)
